htmlの表の幅を調整したい

幅を指定しない状態だと見づらいものになりました。
thのパラメータ設定をこちらのページ[1]を参考に実装しました。 

th.width20{
    width: 20%;
}
th.width40{
    width: 40%;
}
<table>
    <tr>
        <th class="width20">20%</th>
        <th class="width40">40%</th>
        <th class="width40">40%</th>
    </tr>
    <tr>
        <td>20%</td>
        <td>40%</td>
        <td>40%</td>
    </tr>
</table>
20% 40% 40%
20% 40%

styleはcssで定義しました

が、どうにもうまくいかず。 拡大縮小をすると割合が変なことになってしまいます。 ……と思っていたらtableの設定を忘れていました。
styleに追加します。 

table.fixWidth{
    width: 100%;
    table-layout: fixed;
} 
<table class="fixWidth">
    ～以下省略～
20% 40% 40%
20% 40%

使いやすいかどうかはおいておいて、やりたいことができました。
