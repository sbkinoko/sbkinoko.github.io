htmlの表の幅を調整したい

幅を指定しない状態だと見づらいものになりました。
thのパラメータ設定をこちらのページ[1]を参考に行ったのですが、どうにもうまくいかず。 拡大縮小をすると割合が変なことになってしまいます……。 


    <style>
        th.width20{
            width: 20%;
        }
        th.width40{
            width: 40%;
        } 
    </style>



    <table>
        <tr><th class="width20">場所</th><th class="width40">できること</th><th class="width40">gif</th></tr>
        <tr><td>定義場所</td><td>使用箇所を表示できる<br>クリックすると対象にジャンプ</td><th><img src="ctrl_1.gif" width="auto"></th></tr>
    </table>

……tableの設定を忘れていました。
styleに追加します。 


        table{
            width: 100%;
            table-layout: fixed;
        }

表はいい感じになりました。
それはそれとして、gifのサイズがおかしなことになっているので修正します。
widthをautoから100%へ変更です。 


            <tr><td>定義場所</td><td>使用箇所を表示できる<br>クリックすると対象にジャンプ</td><th><img src="ctrl_1.gif" width="100%"></th></tr>

使いやすいかどうかはおいておいて、やりたいことができました。

参考

  1. 『CSS: table（テーブル）の列幅の割合を指定する』　https://step-learn.com/article/html-css/031-table-width-percent.html
