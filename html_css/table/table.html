htmlの表の幅を調整したい

幅を設定しない状態だと、以下の様になり非常に見づらくなってしまいました。

thのパラメータ設定をこちらのページ[1]を参考に行ったのですが、どうにもうまくいかず。 拡大縮小をすると割合が変なことになってしまいます……。 



        
    <style>
        th.width20{
            width: 20%;
        }
        th.width40{
            width: 40%;
        } 
    </style>

    <table>
        <tr><th class="width20">場所</th><th class="width40">できること</th><th class="width40">gif</th></tr>
        <tr><td>定義場所</td><td>使用箇所を表示できる<br>クリックすると対象にジャンプ</td><th><img src="ctrl_1.gif" width="auto"></th></tr>
    </table>

そこでこちらのサイト[2]を参考にtableのstyleを定義しました。 拡大縮小しても画面に対してtableの割合が固定されるようになりました。



それはそれとして、gifのサイズがおかしなことになっているので修正しました
